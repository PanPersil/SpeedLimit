# SpeedLimit — плагин ограничения скорости игроков

**SpeedLimit** — плагин для Minecraft‑серверов, который ограничивает скорость передвижения игроков, полёта на элитрах и обычного полёта.  
Он помогает предотвращать читерство, слишком быстрые перемещения и позволяет тонко настраивать скорость для разных режимов: мини‑игр, PvP и ролевых серверов.

---

## 📝 Описание работы

Плагин отслеживает скорость движения игроков и сравнивает её с заранее заданными лимитами.  
Отдельно контролируется скорость ходьбы и бега, поведение на льду, прыжки/перемещение в воздухе и скорость полёта на элитрах.  
При постоянных нарушениях заданных ограничений плагин может автоматически кикать игрока с сервера за слишком быстрое перемещение.  
Также реализован «grace‑период» после получения урона, когда игроку временно разрешается немного более высокая скорость, чтобы избежать ложных срабатываний.

---

## 🧪 Команды

Все команды плагина начинаются с префикса `/speedlimit` и используются для управления конфигурацией и отладкой:
 
- `/speedlimit reload`  
  Перезагружает конфигурационный файл плагина без перезапуска сервера.  
  Используется для применения изменений в настройках скорости и параметрах защиты.

- `/speedlimit`  
  Отображает нынешний статус отладки. 

- `/speedlimit debug`  
  Меняет текущий статус отладки и отображает в чат нынешнее состояние. 

- `/speedlimit setdebug true`  
  Включает режим отладки.  
  Плагин начинает выводить подробную диагностическую информацию в консоль, что помогает при настройке и поиске проблем.

- `/speedlimit setdebug false`  
  Отключает режим отладки.  
  После этого в консоль больше не выводятся сообщения отладки.

Состояние включённой или отключённой отладки сохраняется в конфигурационном файле плагина.

---

## 🔐 Права доступа (Permissions)

Для тонкой настройки доступа к функциям плагина используются следующие permissions:

- `speedlimit.admin`  
  Даёт игроку право пользоваться административными командами плагина. 
  Рекомендуется выдавать только доверенным игрокам или персоналу сервера.

- `speedlimit.bypass`  
  Позволяет игроку игнорировать лимиты скорости.  
  Игрок с этим правом не будет проверяться плагином и не будет получать наказание за превышение скорости (удобно для администраторов, модераторов или технического персонала).

## 💾 Конфигурационный файл

Конфигурационный файл плагина содержит:

- Лимиты скорости для разных состояний (ходьба, воздух, лёд, элитры и т.п.).  
- Параметры защиты, включая период «иммунитета» после получения урона и ограничение скорости в этот период.  
- Лимит количества нарушений перед киком игрока.  
- Флаг включённого/выключенного режима отладки, чтобы состояние debug сохранялось между перезапусками.

## ⚙️ Дополнительная настройка

Если вы используете ядро **Paper** или схожие сборки, в которых уже есть встроенная проверка на полёт, а также сторонние плагины по типу **AnarchyExploitFixes**, рекомендуется отключить в них проверку на полёт.  
Наш плагин реализует проверку полёта и скорости заметно точнее и гибче, поэтому использование дублирующих анти‑флай систем может привести к конфликтам и ложным срабатываниям.
