# SpeedLimit
Плагин для ограничения скорости передвижения игроков, ограничения скорости полёта на элитрах, ограничение полёта игроков
В плагине есть несколько команд, которые используются для логирования скорости игроков в консоль сервера, по дефолту её можно вводить только в консоль сервера или могут вводить игроки, владеющие правами администратора
/speedlimit debug - включение/отключение отладки
/speedlimit setdebug true - включение отладки
/speedlimit setdebug false - отключение отладки
Плагин тестировался только на версии 1.12.2 с использованием java 8, работу с другими версиями обещать не могу
